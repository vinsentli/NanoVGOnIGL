# NanovgOnIGL

NanovgOnIGL是将[nanovg](https://github.com/memononen/nanovg)与[igl](https://github.com/facebook/igl)相结合而实现的项目，使nanovg能够运行在Opengl, Metal，Vulkan之上。

Support Platforms : MacOS, Windows, Linux, iOS, Android.
